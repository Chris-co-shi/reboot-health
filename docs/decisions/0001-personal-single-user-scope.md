# 0001 单人个人使用范围

## 状态

已确认；访问边界由 [0008](0008-m25a-flutter-agent-device-bootstrap.md) 扩展。

## 决策

本项目当前只服务用户个人，不建设公开多用户产品、注册运营体系、多租户、支付或 SaaS 能力。

单用户不等于无认证。跨 iOS、Android、macOS 和 Windows 使用时，采用私有设备身份和配对机制保护接口，但不扩展为完整 IAM、角色权限或组织模型。

## 原因

当前目标是完成健康、减脂、规律训练和 AI 教练闭环，而不是建设通用账号平台。

## 影响

- 第一阶段只存在一个默认用户边界。
- 新模型保留 `userId`，用于数据归属、导出和未来演进。
- 每台客户端设备拥有独立 `deviceId` 和可撤销身份。
- 不实现公开注册、密码找回、组织、角色和租户管理。
- 设备认证的具体决策见 ADR 0008。
