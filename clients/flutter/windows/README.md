> 当前目录保留 Flutter Windows 工程基础；Windows 不是当前正式用户客户端交付目标。

## Verify

在 Windows 开发机安装 Flutter SDK 与 Visual Studio Desktop C++ workload 后，从 `clients/flutter` 执行：

```powershell
flutter doctor -v
flutter build windows --debug
```

macOS 环境不能替代真实 Windows 构建验收。
